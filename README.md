# Boilerplate para Landing Pages

Este é um **boilerplate** para criar **landing pages** de forma rápida e eficiente, utilizando **React** e **Tailwind CSS**.

## Tecnologias Utilizadas

- **React**: Biblioteca JavaScript para criar interfaces de usuário.
- **Tailwind CSS**: Framework CSS para construção de interfaces de forma rápida e customizável.
- **Lucide React**: Conjunto de ícones para interfaces.
- **TypeScript**: Superset do JavaScript para adicionar tipagem estática ao projeto.
- **Vite**: Ferramenta de build e desenvolvimento rápido para projetos em React.

## Dependências

- **lucide-react**: Biblioteca de ícones.
- **react**: Biblioteca principal para criação de componentes de UI.
- **react-dom**: Biblioteca para manipulação do DOM no React.

## Dependências de Desenvolvimento

- **@vitejs/plugin-react**: Plugin oficial para integrar React ao Vite.
- **eslint**: Ferramenta de linting para garantir a qualidade do código.
- **tailwindcss**: Framework CSS para estilização.
- **postcss**: Processador CSS para otimização e compatibilidade entre navegadores.
- **autoprefixer**: Ferramenta para adicionar prefixos automaticamente a CSS.
- **typescript**: Suporte a tipagem estática no JavaScript.

## Scripts

- **`npm run dev`**: Inicia o servidor de desenvolvimento utilizando o Vite.
- **`npm run build`**: Compila o projeto para produção utilizando o TypeScript e o Vite.
- **`npm run lint`**: Executa o linting no código com o ESLint.
- **`npm run preview`**: Exibe uma versão prévia do projeto após a build.

## Como Iniciar

1. Clone o repositório:
   ```bash
   git clone <url-do-repositorio>
2. Instale as dependências:
   ```bash
   npm install
3. Inicie o servidor de desenvolvimento:
   ```bash
   npm run dev
